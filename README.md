# Test project

### Installation

Project requires [Node.js](https://nodejs.org/) to run.

You need Gulp installed globally:

```sh
$ npm i -g gulp
```

```sh
$ git clone [https://github.com/divizionbyzero/forte.git] https://github.com/divizionbyzero/forte.git
$ cd forte
$ npm i -d
```
