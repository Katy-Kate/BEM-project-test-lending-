# Test project

### Installation

Project requires [Node.js](https://nodejs.org/) to run.

You need Gulp installed globally:

```sh
$ npm i -g gulp
```

```sh
$ git clone [https://github.com/Katy-Kate/BEM-project-test-lending-.git]
$ cd forte
$ npm i -d
$ gulp watch (if you want to change something)
```
